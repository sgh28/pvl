## Steps to run
* Open 2 terminals
* In first terminal
    * cd client
    * npm i
    * npm start
* In second terminal
    * cd server
    * npm i
    * node index.js

## Prerequisites
* nodejs should be installed
* mongodb should be installed 
* database has to be created with name DB
